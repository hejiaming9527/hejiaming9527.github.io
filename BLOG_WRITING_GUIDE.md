# 博客分类与标签使用规范

## 分类结构

每篇文章选择一个主分类，可按需要增加一个子分类。

| 主分类 | 推荐子分类 | 适用内容 |
| --- | --- | --- |
| 项目经历 | 嵌入式开发、硬件设计、人工智能、Web 开发 | 课程设计、作品复盘、功能迭代 |
| 工作记录 | 制造工程、精益生产、质量管理、职业成长 | 工作方法、现场复盘、职业体会 |
| 学习成长 | 专业学习、工具实践、读书笔记、考试记录 | 课程笔记、技术学习、工具教程 |
| 生活随笔 | 日常、校园、旅行、摄影 | 生活记录与个人思考 |

不要创建同义分类，例如“工作”“工作笔记”“职场记录”应统一使用“工作记录”。

## 标签规则

- 每篇文章建议使用 3 到 7 个标签。
- 标签描述具体技术、工具或主题，分类描述文章属于哪个内容领域。
- 技术名称统一大小写，例如 `STM32`、`TensorFlow`、`MySQL`、`Vue`。
- 合作项目统一添加 `课程设计` 和 `小组合作`，不要使用 `独立开发`。

推荐标签词表：

- 嵌入式：`STM32`、`C语言`、`UART`、`I2C`、`OLED`、`语音识别`
- 硬件：`Proteus`、`PCB`、`BOM`、`焊接`、`音频电路`
- 软件与数据：`Python`、`TensorFlow`、`MySQL`、`Linux`、`Vue`
- 制造工作：`精益生产`、`5S`、`SOP`、`IE改善`、`质量管理`
- 生活成长：`生活`、`校园`、`读书`、`求职`、`职业成长`

## 新文章模板

在 `_posts` 目录创建 `YYYY-MM-DD-english-slug.md`：

```yaml
---
title: 文章标题
date: 2026-07-18 20:00:00 +0800
categories: [工作记录, 制造工程]
tags: [精益生产, SOP, 职业成长]
description: 用一句话说明文章内容。
author: 何嘉明
toc: true
comments: true
pin: false
---
```

保存并部署后，Jekyll 会自动将文章加入分类、标签和归档页面，并使用统一的文章详情模板。
