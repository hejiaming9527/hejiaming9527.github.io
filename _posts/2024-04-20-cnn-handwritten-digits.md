## 项目说明

这是大学课程设计，由小组合作完成，并非个人独立开发。本文根据本人参与的数据处理、模型训练和结果分析工作进行回顾整理。

## 项目目的

- 理解卷积神经网络从数据输入、特征提取到分类输出的基本流程。
- 使用 TensorFlow 完成训练、验证及超参数调整。
- 对比基础 CNN 与 AlexNet 迁移学习思路，观察模型复杂度和训练效果的差异。

## 项目背景

手写数字识别数据规模适中、评价方式直观，适合作为深度学习入门项目。小组使用 Python 与 TensorFlow（GPU）搭建包含 3 个卷积层和 3 个池化层的 CNN，并尝试引入 AlexNet 预训练模型开展迁移学习实验。

## 分工与本人参与

项目由小组共同完成。本人参与数据预处理、CNN 结构搭建、训练参数调整、结果记录和迁移学习实验；成员共同完成资料调研、模型验证和课程汇报。

## 技术方案

| 环节 | 方案 |
| --- | --- |
| 开发语言 | Python |
| 深度学习框架 | TensorFlow（GPU） |
| 基础模型 | 3 个卷积层与 3 个池化层组成的 CNN |
| 对比实验 | AlexNet 预训练模型迁移学习 |
| 评价方式 | 训练集与验证集损失、分类准确情况、错误样本观察 |

## 实现功能

- 对数字图片进行尺寸、格式和标签处理。
- 训练 CNN 并保存阶段性结果。
- 使用验证数据检查模型的分类表现。
- 开展 AlexNet 迁移学习对比实验。
- 输出预测类别并观察典型错误样本。

## 遇到的困难与处理方法

### 训练速度和显存占用不稳定

模型训练初期批量大小设置偏大，影响运行稳定性。处理时逐步调整 batch size，并固定输入尺寸与数据类型；先用少量数据验证完整流程，再扩大训练规模，减少反复等待后才发现代码问题的情况。

### 训练效果提升但验证表现波动

只观察训练集容易忽略过拟合。小组将训练和验证指标分开记录，调整训练轮次，并通过错误样本观察模型容易混淆的数字形态，而不是只追求训练指标继续上升。

### 迁移学习模型与输入数据不匹配

AlexNet 的输入形式与手写数字图片存在差异。实验中统一调整图片尺寸和通道格式，明确冻结与训练的网络层，并将迁移学习作为对比方案，而不是直接宣称一定优于较小的基础 CNN。

## 回顾性工作日志

> 以下日志根据课程设计过程回顾整理，用于展示工作阶段，不是当时逐日保存的原始训练记录。

1. **环境与数据阶段：**确认 Python、TensorFlow 和 GPU 环境，完成数据读取、标签检查和可视化抽样。
2. **基础模型阶段：**搭建卷积、池化和分类层，以小批量数据验证前向计算与训练流程。
3. **训练调优阶段：**调整批量大小、训练轮次等参数，分别记录训练集和验证集表现。
4. **问题分析阶段：**检查错误分类样本，观察相似笔画、图片偏移等情况对结果的影响。
5. **迁移学习阶段：**处理 AlexNet 输入格式，完成预训练模型的对比实验。
6. **成果整理阶段：**汇总模型结构、训练过程和实验现象，完成小组课程设计汇报材料。

## 项目总结

该项目帮助小组建立了完整的机器学习实验意识：先保证数据和流程正确，再讨论模型结构和参数；同时必须区分训练表现与泛化表现，并如实记录不同方案的适用条件。
